Comprehensive Security

Enterprise-grade security features

Your data and privacy are our top priority. LearnLoop implements multiple layers of security to protect your information.

  • Password Security - PBKDF2-SHA256 hashing with minimum 8 characters, letters, and numbers
  • HTTP Security Headers - X-Frame-Options, X-Content-Type-Options, CSP, HSTS
  • Session Security - Secure cookies, HttpOnly, SameSite, 7-day lifetime, 24-hour inactivity expiry
  • Rate Limiting - 5 login attempts per 15 min, 3 registrations per hour
  • Input Validation - Email validation, HTML tag removal, SQL injection prevention
  • File Upload Security - Type whitelist, size limits, filename sanitization
  • HTTPS Enforcement - Strict-Transport-Security in production
  • XSS Protection - Content Security Policy and input sanitization
